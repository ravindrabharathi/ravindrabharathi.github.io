Machine Learning projects

1. tf_utils :
Utilities for Tensorflow
A set of utilities to train a Convolutional Neural Network using Tensorflow. This supports creation of TfRecords using links from from Cifar10 and Cifar100 project site , Creation of Data pipeline from tfrecords using tf.Data Dataset , Augmentation strategies like cutout , random-pad-crop, Plotting misclassified images and plotting confusion matrix to see which classes the model mistakes for another. This is a work in progress, and I intend to add more datasets and features.
